# Lotics Workflows

Ready-to-use automation workflows for [Lotics](https://lotics.ai) — the operations platform that lets teams build apps, automate workflows, and manage data with an AI assistant.

## What's here

Each workflow includes the problem it solves, a data model, example prompts you can say to the Lotics AI assistant, and the workflow definition.

| Workflow | Use case |
|----------|----------|
| [Gate Receipt Automation](examples/gate_receipt_automation.md) | Automate gate-in/gate-out documentation at container depots |

## How to use these

1. Open [Lotics](https://lotics.ai) and start a conversation with the AI assistant
2. Copy the example prompts from any workflow
3. The assistant will create the tables, fields, and automation for you

## About Lotics

Lotics is an operations platform for teams that run on spreadsheets, paper forms, and manual processes. Build custom apps, automate workflows, and manage structured data — all through conversation with an AI assistant.
